About This Project

This project presents a Heart Disease Analysis Dashboard aimed at understanding major risk factors influencing cardiovascular health.

Heart disease remains one of the leading causes of death worldwide. Through interactive data visualization, this dashboard highlights patterns related to:

  • Age distribution
  • Gender comparison
  • Blood Pressure levels
  • Cholesterol impact
  • Diabetes-related risks
Tools Used:
  • Flask (Python Web Framework)
  • Bootstrap (Responsive UI Design)
  • Tableau Public (Data Visualization)
